Usage with Kohya-sd-script   v260430

To easily use these Emo series with Kohya-sd-script,
simply place this folder as-is into the "sd-scripts" folder within your Kohya-sd-script installation:

sd-script/optimizer

With this setup,

    --optimizer_type=optimizer.emosens.EmoSens
    --optimizer_type=optimizer.emoairy.EmoAiry
    --optimizer_type=optimizer.emocats.EmoCats
    --optimizer_type=optimizer.emotion.EmoTion
    --optimizer_type=optimizer.emovoid.EmoVoid
    
You can utilize each optimizer by simply specifying one of the above.

|||Usage options|||
    ●FFT-mode on：

    ●Shadow off:
    use_shadow=False
    ●notify off:

    shadow-system: on (option)

    Early Stop indicator ensures no one misses the “learning stop” timing
    The following code displays “✨[READY TO STOP]✨” during learning
    This measures the reliability of the loss signal when the emotion scalar reaches its minimum value and the dNR base also hits its minimum
    Once displayed, it signals that all learnable information has been absorbed—you can stop learning anytime
    Don't rush to stop immediately; instead, try using the saved data from just before and after the light came on (the data closest to the moment the light came on) as your top priority.

        if abs(scalar) <= self.stop_scalar and abs(Noise_base - d_base) <= self.stop_dNRsub:
            if not self.should_stop:
                self.emoScope *= 0.1      # ユーザー意思を目的の収束へ整える
            self.should_stop = True       # 💡 外部からこれを見て判断可
            if self.notify:               # 💡 収束・安定の「お知らせ」
                print(f"✨[READY TO STOP]✨")
        else:
            self.emoScope = self._init_lr # 誤判定はユーザー意思を再反映する
            self.should_stop = False      # 💡 誤判定などの取り消し

    This notification is normally enabled, but you can disable it if you don't need it.
